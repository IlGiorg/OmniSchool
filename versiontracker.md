# OmniSchool Versions
### June 2024 - Legacy 1
OmniSchool was first conceived as OSMAP and the version now marked as "Legacy 1" is made.
Only rudimental hardcoded login, and everything was stored in localStorage.

### July 2024 - Legacy 2
Structure was added with clear differences between admin, student and teacher file folders to differentiate physically the files.
PHP is introduced as "beta" and JSON storage was attempted to store attendance and conduct records.
Login system logic was redesigned

### September 2024 - alpha 1
OSMAP was completely restructured and restarted to be based on PHP and a MYSQL relational database with different tables for students, teachers, conduct records, attendance, administrators etc



### April 2025 - xxx
OSMAP becomes OmniSchool
The first iteration of basic branding is implemented in the homepage.
UI improvements and some UX improvements were attempted
Grades function was added & Conduct overhaul

### June 2025 - xxx
Attendance was implemented

### January 2026 - Parent Portal
The parent portal was created
Cleanup for database paths started. Only a few files were updated

### February 2026 - Version 1
Giorgio deemed OmniSchool fully functional and "just enough to call it an actual version 1"
